- download the Expo Go app
- clone the repo, and run:
   - npm install
   - npx expo start

## Overview
You adopted a pet! Start a study timer to earn food to feed your pet. The longer the timer the better the food and the happier your pet gets.

### User flow
* Choose your pet
* Name your pet
* Pick a timer
* Keep your pet happy!
